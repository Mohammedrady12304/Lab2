for remote repos :
git push origin --delete dev
git push origin --delete test


for local :
git branch -d dev
git branch -d test


and the head should be on the main



how to list tags :
git tag


how to delete tag locally :
git tag -d v1.7


how to delete tag remotely :

git push origin --delete v1.7
